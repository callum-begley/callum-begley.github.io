What is JavaScript and how does it relate to HTML and CSS

10 April 2025

What is JavaScript and its relationship to HTML and CSS?

Imagine a website is like an aeroplane. HTML is the structure of the plane, where the wings are, where the cockpit is. CSS is the design of the plane, what colour it is and the patterns on the seats. Javascript is the engine and all the mechanics of the plane that you don't see but are crucial to making the plane fly. Controlling the wings rudders, maintaining altitude and deploying the landing gear. Javascript working behind the scenes is like magic, but coding it is the opposite. Javascript adds interactivity to your webpage and allows for elements to change without having to refresh the page. HTML CSS and JavaScript all need to work together to create an advanced modern webpage.

Control flow and loops

JavaScript by default will run from top to bottom, running through the code line by line. Control Flow lets the program make decisions when to execute the code. It does this with conditional statements and loops.

Conditional statements

Conditional statements are checks to see wether a condition is true or false. Below is an example to see wether you need to take an umbrella with you. 

          if (its raining outside) { 

                grab umbrella
                leave

              } else if (if it looks like it might rain){

                grab umbrella
                leave

              } else {

                put on a hat
                leave

              }

Loops

Loops let you repeat an action a fixed number of times or until a condition is met. a For loop is set with a counter (i = 0) and the condition is how many times to loop through (i < 10) then the count is updated (i++). 

      for (let i = 0; i < 10; i++) { 

              loop through 10 times

            }

You have to be careful to give a condition that will end or you may create an infinite loop that could crash your computer.

Do not do the below loop. As eating snacks will never make boredom false you will end up snacking infinitely. 

          while (boredom is true) { 
    
                  eat snacks
    
                }

What is the DOM?

The DOM, or Document Object Model is a representation of the webpage that lets JavaScript interact with the style, structure and content of the page. It is a tree like structure made up of the elements of the HTML document, with other elements and attributes nested in. (See image for a visual representation.) These elements can be modified, added and removed using JavaScript. The DOM also has the ability to react to user events like clicks ot key presses, making your website more interactive and responsive.

Below are some examples of how JavaScript uses the DOM to interact with the HTML elements:

document.querySelectorAll("p") will access all the p elements

document.getElementById("this") will access the element with id="this"

document.getElementByClassName("that") will access all the elements with class="that"

Accessing data from arrays and objects

Arrays

Arrays are a list of values in one variable. These values can be accessed by referring to their index numbers. It is important to note, arrays start counting at 0. 

                           0        1        2
        const drinks = ["coke", "sprite", "fanta"];
          let myDrink = drinks[0];

myDrink selects the first item in the array (0), so my drink will be a coke. If it was drinks[2] my drink would be a fanta.


Objects

Objects are variables that hold key and value pairs. below is an example object of a dog:

        const dog1 = {
            name: "Chump", 
            breed: "Pug", 
            cuteness: 10,
          }

The keys are the name, breed and cuteness, with the values detailed beside them. Objects properties can be accessed by objectName.keyName or objectName["keyName"]

dog1.name will give you a string of "Chump"dog1.cuteness will give you the number 10


What are functions? Why are they helpful

A function is a block of code that does a particular task. Functions will only run if they are called. Functions are made by using the keyword function, then the name followed by brackets. They can also take in parameters inside the brackets.

        function exampleFunction() {
          execute this code
        }

This will only run if it is called by exampleFunction() outright or inside of another function.

Functions are very helpful as they save writing out the same code over and over again. If you need to execute the same lines of codes multiple times, create a function and call to it with just the name and brackets whenever it is needed.
