What is JavaScript and its relationship to HTML and CSS?

Imagine a website is like an aeroplane. HTML is the structure of the plane, where the wings are, where the cockpit is. CSS is the design of the plane, what colour it is and the patterns on the seats. Javascript is the engine and all the mechanics of the plane that you don't see but are crucial to making the plane fly. Controlling the wings rudders, maintaining altitude and deploying the landing gear. Javascript working behind the scenes is like magic, but coding it is the opposite. Javascript adds interactivity to your webpage and allows for elements to change without having to refresh the page. HTML CSS and JavaScript all need to work together to create an advanced modern webpage.

Control flow and loops

JavaScript by default will run from top to bottom, running through the code line by line. Control Flow lets the program make decisions when to execute the code. conditional statements and loops.

Conditional statements are checks to see wether a condition is true or false.
